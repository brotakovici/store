store
=====

A store system that will include (hopefully) the following:
   
1. Cart system
  * add to cart
  * remove from cart

2. Wishlist
  * in stock modifier
  * promotions
        
3. Log-in system
  * register
  * log-in 

4. Check-out system
  * choose payment:card/paypal
  * shipping cost calculator
  * payment notifier
  * payment system

5. Product system
  * add product
  * remove product
  * edit product
  * stock
